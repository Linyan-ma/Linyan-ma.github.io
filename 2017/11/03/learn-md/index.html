MarkDown快速入门

发表于 2017-11-03 | 分类于 CODE

学习使用MarkDown基础语法的使用，仅包含最基本的几个符号标识，仅为了能够快速使用MarkDown写blog。

#号表示标题

1-6个#号代表1-6级标题

>号表示引用

满纸荒唐言

* _表示强调

单个*或者_包裹的字体会转换为,两个包裹的字体会转换为

示例

*、+、- 无序列表

示例

  • second

\1. \2. 有序列表

示例

  1. 程序一
  2. 程序二

示例
